    To notify others that you will be away from your email for a period of time, please use the “vacation” program.

    The Setup

    1. Edit the “.vacation.msg” file in your network drive to reflect the vacation message you want sent, using any editor but be sure to save as a plain text file if you are using a word processor. Please see NOTES below if you do not see any files beginning with “.” These are “hidden” files and you may need to adjust your Windows explorer or Macintosh Finder settings.

    2. Open a terminal window on a Unix server (sphinx is recommended) using Putty or Ssh Secure Shell on a Windows PC (these programs are found under Biostat/General in the Windows Program Menu), or Terminal on a Macintosh (which is located under Applications/Utilities).

    3. At the Unix prompt, type “vacation2 enable“.

    At this point you can test your vacation message by sending a message to yourself from a different (possibly a personal email) account or have an associate send you an email.

    NOTE:
    To return to normal email settings, follow steps 2 and 3 but instead type “vacation2 disable“.

    Your correspondents will receive the message you have saved in .vacation.msg the first time they email you after enabling vacation mode. They will not receive the notification in subsequent email attempts until after a predetermined interval.

    DISPLAY HIDDEN FILES

    To display hidden files on a Windows PC:

    In Windows 7, open My Computer, and double click on your Z: drive. At the top of the file window, click Organize, then click Folder and search options, click the View tab, and Select “Show hidden files, folders, and drives”. Also, uncheck “Hide extensions for known filetypes” and click “Apply“. Close the Folder Options window and you will now see all files including previously hidden “dot files” with complete filenames, including extensions like .docx (Office Document) .pdf (Adobe PDF) etc…
