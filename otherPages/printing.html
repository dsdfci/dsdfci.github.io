    LC9 Printers Instructions

    Please let an admin know if the toner is low or if paper is running out. If you do decide to replace the toner yourself, please give the old toner cartridge to an admin for proper disposal. Connect your computer or laptop to the PHS wifi or to an ethernet connection to print. Instructions are also located on the printers themselves.

    To install a printer on a Mac on the DFCI network

    Open this document for instructions.

    To install a printer on a Windows machine on the DFCI network

    Click Start Button (bottom left of taskbar) and in the search box, paste \\helo.bcb.local then press the Enter key

    If you get a message the network location cannot be found, even after rebooting, use the following command instead:
    Click Start Button and in the run box, paste \\155.52.47.85 then press the Enter key Double click each printer you’d like to install

    Use BCB\yourjimmyusername and your jimmy password if you are asked to authenticate.

    To remove a printer installed from Tigh (155.52.45.8) on a Windows machine on the DFCI network

    Click Start Button, Control Panel, click View Devices and Printers (may be listed as just Devices and Printers)

    Right click on each printer you want to remove “printername on tigh” (tigh.bcb.local or 155.52.45.8)

    Click Remove Device
