    • Please run all things via sbatchsbatch or srun. Please do not run things on the head node and please do not ssh into the work nodes to run things.

    sbatch

    sbatch /full-path-to-my-script to submit a job
    sbatch –array=0-10 -N 1 myscript to submit a job array with index values between 0 and 10
    sbatch –array=1,5,9 -N 1 myscript to submit a job array with index values of 1, 5 and 9
    sbatch –array=1-7:2 -N 1 myscript to submit a job array with index values between 1 and 7 with a step size of 2 (1,3,5 and 7)

    sbatch options

    You can state your options via the command line or via the submit script. See Sample Submit Scriptpage for more options.

    Run man sbatch for all of the sbatch options and a brief detail on each.

    srun

    srun /full-path-to-my-command to run a command/interactive session on a work node
    srun -N 3 /full-path-to-my-command –pty bash to run the command on 3 nodes using bash – don’t forget the –pty or your command will run 3 times separately per node
    srun -N 2 –ntasks-per-node=4 –pty bash to run the command on 2 nodes with 4 tasks per node = total of 8 tasks
    srun –x11 -p interactive –pty bash to run an X11 session
