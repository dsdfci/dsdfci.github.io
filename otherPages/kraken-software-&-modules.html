    To see what you have loaded

    $ module list
    Currently Loaded Modulefiles:
    1) gcc/8.2.0 2) slurm/18.08.4

    To load a module

    $ module load lapack/gcc/64/3.8.0

    $ module list
    Currently Loaded Modulefiles:
    1) gcc/8.2.0 2) slurm/18.08.4 3) lapack/gcc/64/3.8.0

    To unload a module

    $ module unload lapack/gcc/64/3.8.0

    $ module list
    Currently Loaded Modulefiles:
    1) gcc/8.2.0 2) slurm/18.08.4
